# Ansible介绍



## 什么是Ansible?


Ansilbe是一个部署一群远程主机的工具。远程的主机可以是本地或者远程的虚拟机，也可以是远程的物理机。

## Ansible能做什么？


Ansilbe通过SSH协议进行管理节点和远程节点之间的通信。理论上说管理员通过ssh到一台远程主机上能做的操作Ansible都可以做。

包括：
* 拷贝文件
* 安装包
* 起服务
* ...
